# Sistema de Controle de Acesso para Cofre Digital com ATmega328P
O projeto consiste em uma interface de segurança para um cofre eletrônico simulado, controlado por um microcontrolador ATmega328P. O sistema gerencia a abertura e o fechamento de uma tranca eletrônica baseada em validação de senha, monitoramento de soensres de porta e feedback visual ao usuário via LCD e displays de 7 segmentos.
